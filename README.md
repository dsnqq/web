# 🧠 Моя база знаний

Добро пожаловать в мой Obsidian-второй мозг. Здесь я собираю, структурирую и развиваю свои знания в веб-разработке и смежных направлениях.

## 📋 О проекте

Этот репозиторий — моя личная база знаний, построенная на принципах Zettelkasten и PARA. Все заметки написаны в Markdown и синхронизируются через GitLab.

**Основная цель:** создать живую, связанную систему знаний, которая растёт вместе с моим опытом и помогает быстро находить нужную информацию.

## 🗂️ Структура
📁 Inbox # Временное хранилище (быстрые заметки, идеи)  
📁 Projects # Активные проекты (каждый проект — отдельная папка)  
📁 Areas # Области знаний (Web, Backend, DevOps и т.д.)  
📁 Resources # Постоянные материалы (книги, статьи, видео)  
📁 Archive # Завершённое и устаревшее  
📁 Templates # Шаблоны для новых заметок  
📁 People # Заметки о людях
