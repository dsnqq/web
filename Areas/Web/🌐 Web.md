**Назначение:** центральная область для всех знаний, связанных с веб-разработкой. Здесь собираю заметки по фронтенду, инструментам, концепциям и смежным технологиям. Это живая база, которая растёт вместе с моим опытом.

## Правила работы

- Каждая заметка должна относиться к одной из подпапок. Если тема новая - создать подпапку и описать её в этом README.
- Использовать осмысленные имена файлов: `kebab-case.md` (например, `css-grid-guide.md`).
- Активно использовать внутренние ссылки (`[[...]]`) для связывания связанных концепций (например, из `JavaScript/` на `Concepts/event-loop.md`).
- Добавлять изображения, примеры кода и внешние ссылки для наглядности.

## Теги

- **Обязательный тег:** `#WEB` для всех заметок в этой папке.
- **Дополнительные подтеги:** `#WEB/HTML`, `#WEB/CSS`, `#WEB/JavaScript` и т.д. (по подпапкам).
- Для фреймворков: `#WEB/React`, `#WEB/Vue`.
- Для инструментов: `#WEB/Tools`.
- Можно добавлять теги уровня сложности: `#beginner`, `#intermediate`, `#advanced`.
