# 📥 Inbox

**Назначение:** временное хранилище для быстрых заметок, идей, ссылок и любой входящей информации. Здесь можно писать быстро и без структуры.

## Правила работы

- Не думать о форматировании - просто фиксировать мысль.
- Желательно добавлять тег `#inbox`, чтобы легко фильтровать.
- Регулярно (раз в день‑два) просматривать заметки и **обрабатывать**:
  - Удалять ненужное.
  - Переносить полезное в соответствующие папки (`Projects`, `Areas`, `Resources`).
  - Превращать идеи в задачи или проекты.
- Стараться не держать заметки в Inbox дольше недели — иначе они могут терять актуальность.
