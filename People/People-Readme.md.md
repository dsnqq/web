
# 👥 People

**Назначение:** заметки о людях — коллегах, менторах, клиентах, партнёрах и других значимых контактах. Помогает помнить контекст общения, детали и связи.

## Правила работы

- **Один человек = одна заметка**. Называть файл по формату `Фамилия Имя` или `Никнейм` (например, `Иванов Иван.md`).
- В заметке фиксируйте:
  - Контакты (телефон, email, соцсети — опционально, без избыточности).
  - Контекст знакомства (проект, мероприятие, откуда).
  - Даты встреч или "созвонов" и краткие итоги.
  - Личные детали (увлечения, дни рождения — только уместное).
- Использовать теги для категоризации: `#коллега`, `#ментор`, `#клиент`, `#нетворкинг`.
- Связывайте с проектами (`Projects`), областями (`Areas`) или ресурсами, где этот человек упоминается.
- Регулярно обновляйте после встреч — заметка должна быть живой.
