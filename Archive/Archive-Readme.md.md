# 🗃️ Archive

**Назначение:** хранение завершённых проектов, устаревших заметок и всего, что больше не активно, но может пригодиться для справки или истории.

## Правила работы

- Сюда перемещаются папки/заметки из `Projects` после завершения работы, а также устаревшие материалы из `Areas` и `Resources`.
- **Не удалять ничего сразу** - архив даёт безопасность и возможность вернуться.
- Структура внутри архива повторяет оригинальную (например, `Projects/НазваниеПроекта`, `Areas/Web/...`), чтобы легко ориентироваться.
- Раз в полгода можно проводить ревью: удалять совсем ненужное, оставлять только ценное.
